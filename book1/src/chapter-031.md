# 第31章 百年器成功被抢-2



洛宾从稳固后的「**交错幕景**」来到杨子藏身旁。

「**怎么样，有没有好东西。**」

「**有张弓好像还不错。**」 杨子藏从缝隙处抬头道。

哪怕是普通的弓，没有手艺也做不出像样的，这张硬弓明显做得还比较精良。 只是在时序的世界不知能有多大作用。

两人合力撬开了门，里面除了弓与斧，床下还有两个木箱。

拉出来一看，洛宾手上的木箱内是御寒的被褥和两件大衣。 而杨子藏手中的另一只木箱，则是一箱沉重的矿物。

「**这矿石，可以在总部那边换点东西。 哈哈，咱们运气不错。**」

......

带著几样东西回到梁喜那边时，老远就闻到了烤肉的味道。

虽然味道不怎么样，但在这样的环境下吃烤肉，几人还是感觉别有番风情。

「**这个时候要是有啤酒就好了。**」 洛宾哈哈笑道。

两个小时后，一脸雀跃的梁喜背著硬弓，跟著手持染红长枪的杨子藏从破烂衣柜中出来。

「**哇哇哇，好轻松就有接近八年时序！**」

「**有时序战器在，一个 c 级交错地居然三四个小时就搞定了。**」

洛宾从裂缝中满意的走出。

「**不愧是百年器，平时大伙儿光是等时序重新增幅就很花时间。**」

「**是啊，一把时序战器的确节约非常多时间。**」

洛宾抛著手中的一块表面坑坑洼洼的粗糙球形水晶石，里面映现著森林的模样。 这是彻底稳固所有的区域后，出现在正中心的物品。

它里面蕴含著完整的地形外貌，以及 20N 时序。

在三人都出来没多久，这裂缝内的光芒就变得暗淡，最终回归普通的黑暗。

「**它以后可以作为我们的备用据点。**」

洛宾道，说罢他在如同荒废十数年的房间内翻找起来。

在消灭恶魇之卵后，屋内物品大多都变得及其陈旧，如同垃圾堆。 它们都缺失了某种精气。

杨子藏在一旁感受著这次的收获，共计 15N。

相比于 b 级交错地它显得很少。

但这比在老鼠巢穴内，面对铺天盖地的怪物们来得轻松多了。

「**所以 c 级交错地是所有人的最爱。 它的频晶价值也是最稳定的，基本都是二三十年时序换一枚。 运气好，还能收获些异界的奇异物品。**」

说到这里，刚从衣柜抽屉找到个陈旧首饰盒的洛宾面色古怪的抬头，看向正琢磨著什么的梁喜。

「**你上个月的 c 级频晶呢？**」

「**古队长拿走了。**」 少年略带失落的叹息道。

他也才知道 c 级频晶价值居然这么高，弱也有弱的好处啊。

只是这个月的频晶，应该又已经送到了古队长那里。

将根金项链揣进兜的洛宾走来拍拍少年肩膀。

「**没事，后面你重新共鸣个队伍就行。**」

四边形Moexl快速展开，几秒后，洛宾转头笑道：「**走，去下一个地方。**」

梁喜变得振奋起来，洛宾这进化后的Moexl，简直就是神技。

......

夕阳虽已坠入地面但天色还很亮堂，夜幕的真正到来还要很久。

越来越多的人聚集到老鼠的怪物巢穴附近，他们务必要使一个蚊子也飞不进去才好。

众人交流，据说曾参与过上次类似战斗的人正在从总部赶来。

......

疲惫的三人再次从一个裂缝中走出来的时候，梁喜瘫软倒在地上。

他脚上有好几处伤口，撕下衣服所包扎的伤口处，布料被鲜血染红。

洛宾小心拆开这孩子的「**绷带**」，伤口已经结疤。

「**真的已经好了？**」

刚才一进入交错地就天旋地转，只因梁喜所站立的平地在变换后，成为了个大洞。 离开两个四边形较远的梁喜，没有维持幕景显现的能力，只有两根线的他很快随变动的幕景地消失。

「**刚才吓死我了。**」 杨子藏道。

「**问题不算大，过会它们就会恢复。**」

梁喜指了指自己已经结疤的部分伤口。

「**我指的是这个吗！**」 杨子藏无语看向这小少年。

梁喜身上满是伤痕，就像被无数刀、枪、抓挠、撕扯后所留下的痕迹。 事实也是如此。

「**唉呀，可惜你的能力了，只能给自己用。**」 洛宾啧啧称奇。

他们都已经知道，阿梁之所以敢一个人进入怪物巢穴，就是因为他自己拥有稀少的，治疗伤势的Moexl。 这Moexl甚至能止饿。

这已经算是很稀有的能力了，却只能对自己使用。

「**对不起...**」少年道，「**可能是Moexl等级太低了。**」

「**笨蛋，两根线又不是你的错，你道什么歉。**」

梁喜势弱的道：「**我听说，Moexl的诞生等级好像还是跟人的精神力有关。**」

他的Moexl只是来自一个小医务室，那还是阿梁和他父亲运气好，上个月在一个屠宰场内部偶然找到的小屋子。

屠宰场里被忽略的小灶间，被锅碗瓢盆挡住的架子下，还有著急救用的医疗物品。

梁喜意外获得了隐藏的医务室能力。

在这个怪异的城市，大型医院，乃至诊所，早就被人瓜分了。

想要在那些地方孕育Moexl，要么花非巨大的代价——材料、金银或时序，要么就只能硬抢，或是等待新区域的诞生。

因此梁喜的运气不能说不好。

「**没关系，以后制作成能力球就行了，就是代价大点。**」

洛宾笑呵呵的安抚，随后看向大汗淋漓的杨子藏。

「**还能坚持吗？**」

年轻人脸色潮红，他点点头。

开玩笑，刚才为了赶时间，两人各自进入不同交错幕景区域，马不停蹄的击杀著里面的猛兽。 最后以「**地貌聚合**」，才让带著梁喜消失的幕景地重新出现。

因此他都快脱力了。

但这趟分配下来，每人也收获了二十年时序。

「**休息休息，我们继续，唉呀，今天真畅快。 要是每天都这样顺当就好了。**」

......

第三处 c 级空间在附近一个小型公园。

远处六七人正进入裂开的地面。

「**这其实是最保险的人数。 而且就算如此，普通人也可能受伤甚至死人。 我们去下个入口吧。**」

就在三人转身离开时，远处一个眼尖的道。

「**老大，刚刚那个人好像是杨子藏。**」

「**杨子藏？ 那个有百年器的家伙？**」

花园缝隙旁，走在最后的曹茂典和小队长余晨停下脚步。

「**就是他手上那把枪，那可是百年器啊老大。**」

「**总部的人应该在关注他吧。**」

「**看样子他们在找交错地，嘿嘿。 总部的关注能有什么用，等他们进入交错地，只要他不展开Moexl，咱们不就有机会。**」

曹茂典的心脏砰砰直跳，满脑子都是那把枪。

余晨眯了眯眼：「**得好好研究下，如果有人一直跟他心灵通话，在交错地内应该可以一试。**」

已下行几步进入裂缝斜坡，背著行囊的青年回过头：

「**真是他？ 何必那么麻烦。 像上次那样，我们只要不让人认不出来不就行了，而且此刻跟外面可是有时差，外面那些人应该不好进行连接。**」

真是天助我也，几人同时想到。

「**对诶，这里面又人人自危，谁敢把自己的寿命交给那被人追杀的家伙。**」

「**到时候随便找个顶包的出来认领抢劫，我们再说从抢劫犯手上帮忙找回了百年器就行了。 老大，咱们可要抓紧机会。**」

余晨几番思索后道：

「**曹老弟你说得对。 这样搞，哪怕总部要，我们也有理由优先购买。 而且我们队伍还没有时序战器，这枪我们可以作为队伍武器直接留下。 走，跟著他们。**」

就在这片加速区域的边缘，一辆改装成房车的卡车中。

——「**这枪... 留下... 走，跟著他们。**」

小电视的喇叭中传来略带背噪的交谈声。

画面显示著三个正在 c 级交错地入口旁讨论抢劫方案的人。

沙发上，男人一脸古怪的看著前方的小巧电视萤幕。

「**既然他们知道在'多余世界'外，我们能有方法远距离关注某人，那为什么还要在外面讨论？**」

黑色制服的他扣了扣脑袋。

「**这些家伙是傻鸟吗？**」

「**估计是，认为你还没进来时间加速区域吧。**」

将头凑过来的男孩子闪动著大眼睛道：

「**而且他们虽然知道我们能远距离关注某人，但又有谁会晓得你的能力是，连被关注者接触过的角色也可以暂时看到呢。**」

曹、余等数人兴致勃勃的从交错地出来，快速实施著方案，看来他们很有经验。

但这些人忘一件事。

他们没有在交错地讨论这次的抢劫方案，时间上更是在杨子藏刚离开不久。

「**呵，希望这些蠢货动作快点，我对杨子藏的关注时间已经只剩不到两小时了。**」

「**燕哥你继续投资咯。**」 少年往口中丢了块小坚果。

[**配角故事演绎告一段落。**]

燕逸胸前Moexl闪动了下。

小萤幕的画面中，自动切回了杨子藏的现场，三人正聊著天穿过片异域风情的圆形红砖广场。

「**今天杨子藏还没有展开过融资Moexl，希望到时候他能再次融资。**」

这位看上去十五六岁，精神奕奕的少年从大沙发上跳了下来，插著腰道：

「**反正没多少时间了，让我去吧。**」

「**这... 那好吧，你小心点别暴露。**」
