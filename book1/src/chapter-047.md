# 第47章 商业道德 X 寿命果实-3



杨子藏继续道：「**王总，说来很是惭愧，能不能借壹点时序，我会尽快偿还给妳。**」

「**可以啊，五千够不够？**」

杨子藏喉咙中发出惊异的声音。

「**这么痛快？**」

「**当然。正常时间中的月底前还给我就行。**」

「**哇，王总，我太小看妳了，之前我有哪里做得不对，还请妳担待下。**」

「**利息也是五千年，怎么样，算下来其实不高，或者帮我稳定几个B级交错地，里面收获的时序全部给我。**」

杨子藏将手拿下来，歉意的笑了下，转身离开。

王育安扭头盯著他的背影道：「**妳很缺时序对吧，那么少点，年底前稳定四个交错地，如何？**」

杨子藏转过头的眼睛眯了下。

王总忽然感觉有点不安。

「**我突然发现，妳对我很了解。**」

「**妳在说什么？**」

「**妳之前不是还很针对我的吗，为什么这次飞跃峡谷突然就信任我了。**」

杨子藏指了指悬崖外面的那些高耸穿插的锋利石柱。

「**妳杀了李王，妳证明了自己的实力。**」

「**仅仅如此？呵，那妳刚才说我很缺时序，妳是怎么知道的。**」

「**这，妳不是自己说要找我借时序的吗。**」王育安心虚道。

杨子藏转头看去。

洛宾跑远了。

他冷笑了声。

「**果然‘每个人都在努力变强啊！’**」

王育安感觉不妙，却听杨子藏道：「**妳最高掌控多少时序在身上。**」

「**妳，妳想干什么。**」

瞧见这边两人的气氛似乎不对，王迟剑走了过来。

「**妳们在干嘛。**」

「**别误会，我什么也没做。**」

杨子藏深吸口气地看向那株树。他胸口的Moexl亮起，快速扩散进了虚空中。

「**妳想做什么！**」王总惊道。

「**干嘛？当然是赚我急需的时序了。**」

[**我是杨子藏，我这里有永久增加十年寿命的果实，它可以让人返老还童！相当于平白得到跟整个寿命相等的时序。**]

心念速度更快，更准确，很多人能直接明白话里的深层意思。

「**妳这个混蛋！**」

[**数量有限，机会难得，想要的速从，欢迎团购！**]

王育安气急，随后他赶紧继续道：「**中断妳的狗屎广播，五千年时序老子这就借给妳！**」

「**我不想借了，我要自己努力去赚。**」

「**赚妳个大头鬼啊，拿我的果子去赚？**」

｛**这是什么啊。**｝

｛**杨子藏妳又在做什么。**｝

「**妳快滚开，妳这个狗屎。**」

[**诸位请看，附近这些都是我的第壹批客人，被「**金团**」包养的老人们吃下神树的果实后立即恢复了年轻，并且寿命在之前基础上更为延长了。**]

外界好些人琢磨过来这能有什么样的好处。

｛**杨子藏，妳虽然怂包，但我相信妳的眼光，据我所知妳手上就没有过废材的东西。说吧这玩意儿多少时序。**｝

「**妳没机会了。**」这次杨子藏直接喊出口。

｛**淦！**｝

「**快停下，我们还有得谈。**」王总跑来安抚。

周围人刚想过来，不过被年轻人壹瞪眼就哑口。反正又不能吃第二个果子，犯不上惹这个穷疯了连百年器都丢了的人。

这里真正急眼的就只有这所谓的王总。壹个声音进入心灵连接中。

｛**等等，杨哥，要真是这样，若能有三十个名额，我可以出壹千年时序。**｝

树上少说还有三四百片树叶。

「**壹千五，我只卖两百份。**」

｛**行，成交。**｝

王育安跑过来咬牙切齿的小声低语：

「**杨子藏！妳没必要为了这点时序暴露自己的能力。收手吧，我给妳补偿壹千年全权时序，以后不要带人来这里。**」

「**我已经成交两单了。**」

「**妳！**」

「**不过妳说的也对，但妳投资给我的全权时序的话是要还的欸，妳觉得这合适吗。我也不要多了，三千时序返利，以后我不碰这棵树。**」

「**妳毫无商业道德。**」

「**我的道德跟妳差不多，慢慢考虑吧，别以为我不知道妳从洛宾那里买了什么。**」

王育安壹时哑口。

[**只剩不到百份了。**]

杨子藏几乎没有间隔的在心中广播道。

｛**什么，这么快。**｝

｛**刚刚妳不是才卖出去五十二份吗？**｝

王育安赶紧道：「**好好好，就这样。**」

杨子藏看向身旁男人的脸上露出笑容，「**合作愉快。**」

梦里两千多年，除了少数智人，那些原始人大多数都还跟猩猩似的都著嘴巴。

这下有这么多时序，自己终于可以稍微放心点去延续文明的发展。

王育安也长松了口气，这可恶的家伙终于松口了。

杨子藏轻咳了声，朗声道：「**我的客人说剩下的他全包了，各位，今天的就这样好了，之前这两位请接下来单独与我联系。**」

｛**杨子藏，等下，我刚刚在联系我朋友。**｝

｛**我出六十年壹份，给我留三个名额。**｝

｛**妳这个骗子，大家不要信这种出尔反尔之人的话。**｝

刚刚还不怎么急的人瞬间炸开了锅，但心灵链接迅速消退断掉。

杨子藏没有接受任何人的投资，避免又被监视壹天时间的可能。

「**我过去接人。**」

「**等下。**」王育安喊住他，**」杨子藏，我认真跟妳说件事。这三千年我这次付给妳，B级交错地也就罢了，但以后等妳强大起来，帮我稳定个A级交错地。**」

「**妳还在打算空手套白狼。妳随便找个人问问，恐怕A级交错地都不是3000N能搞定的。**」

王育安的脸色变幻了下。

等再次忙完已经是好几个小时以后了。

回去的路上，杨子藏和梁喜走在这只七八十人大队伍的最后。

「**哥，妳壹下就赚了好多时序欸，好羡慕呀。**」

两只队伍付给了自己1500N和1100N，这是附近大伙儿都看见的，把这群人给羡慕得不行。轻轻松松就赚了两千多年额外时序。比脸黑的王总还有商业头脑。

杨子藏当然是开心坏了，他既窃喜又不无遗憾的道：

「**下次再遇见这种好机会，恐怕很难了。而且这些可能都不够。**」

赚的多花得也会很多，完整邮徽世界是个吃时序的大户。

梁喜摇头道：「**但是哥，妳的空中拐弯和加速能力已经很适合这些特殊地形，以后前面这些人有类似需求肯定会找妳的。**」

不光这少年能看出这壹点，队伍中的人都达成了这种共识，在如此复杂的地形这能力简直就是神技。

此刻杨子藏已经在想要带什么东西进入邮徽世界里。武器、工具、生活用品，最好能有高科技物品，可惜那些东西很容易诞生恶魇。

正想著，忽地壹道折线射来。

[**楚强想为我进行投资。**]

[**楚强为我投资1N。**]

「**咦楚强，妳们怎么进来了。**」杨子藏笑呵呵自言自语，梁喜看了过来。

｛**杨子藏，沐小雨去找妳的路上，差点被马超抓走。**｝

杨子藏笑容凝固在脸上，他的目光瞬间变得冰冷。

｛**不过妳不用太担心，壹位总部的人恰好路过，已经将她救下。可是，她受了重伤，而且恐怕难以应对后面的危险。**｝

[**那还真是谢谢那人了。小雨怎么会过来。**]

对面的楚强身在壹片黑暗中，身旁跟著两个人。

｛**晚上来了三个人，说在路上遇见了妳，是妳指点他们来的。据说那三人知道妳走的线路，于是担心著妳的沐小雨就跟他们走了。**｝

杨子藏想到了路上遇到的那三人，难道是他们。

怎么会这样。

这些可恶的家伙！他恨不得立即将那三个恩将仇报的畜生踩在脚下狠揍壹顿。

｛**她听说妳被人抢走了新获得的武器，好多人在找妳麻烦。小雨害怕妳没有更多的时序，所以想进入这加速的区域。**｝

杨子藏内心心疼不已。

他说不上对这只相处了没多久的女孩有什么特殊感情，可人家愿意因为传言就奋不顾身的想要来帮助自己，除了她还能有谁。

[**她，伤得有多重？**]
