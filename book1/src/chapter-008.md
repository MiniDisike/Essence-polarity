# 第8章 前往与助救 -2



暴怒的老鼠贴身攻击，这一棍直接甩在它变得怪异的头颅上。

嘭！

粘稠的液体飞溅，老鼠庞大的身躯站立原地，随后往下直直压来。

[**我收获了 39.4N，这都是强大的双...**]

男孩被这堆巨肉压住，他强忍刺鼻臭味，喘著粗气从怪物身下吃力爬出，眼神有些涣散地道：「**谢谢，大家...**」

他主动切断了联系，Moexl自动分配了时序。

随后，梁喜吃力抬头，看向黝黑的通道交接处。

想了下后，他闭目感应了下还若有若无的联系，身上的曲折光线遵循著杨子藏的公开气息，延伸进身旁虚空。

「**大哥哥，在死之前，我要告诉你、一个秘密...**」

......

另一边。 来自那孩子的感应画面消失了。

[**应选者归还了我的投资，并分予了 80%红利，共计约 206.8N。**]

这最后的战斗，只在电光石火之间，随后杨子藏就感知到了反馈。

他脑袋蒙了下，五十六年？

这就得到了五十六年？

「**那个弟弟好厉害，我居然分到了十一年零三个月！ 太夸张了。 这是真的吗？**」

沐小雨有些激动。

她是见杨子藏选择了投资，于是自己也跟著投，没想到仅仅三十年就赚了这么多！

张扬呵呵一笑，说道：「**我也赚到了接近四个月的时间呢。 唉，就是那孩子，不知道后面怎么办。**」

见杨子藏似乎在思考什么，没回应两人，他看向身旁头发花白的刘阳。

「**一寸光阴一寸金，我算是明白了。**」

「**靠。**」 刘阳将脸转到一边去。

刘阳想到，这儿所有人中，就只有自己最短命。

眼看身旁两个家伙已经投资两次，自己每次却只能赚到几天。

不过能增加几天的时间也很好了，说不定转机就在那几天之中。

刘阳感慨道：

「**难怪阿梁那小子敢去那种地方，虽然危险，但仅仅两只老鼠怪物就能获得这样多的时序，真是赚啊。**」

「**这是因为只有咱们投资，而且他又愿意分出八成的时序，要是有别人参与，随便来个几十年，咱们的红利就少得很了。**」

「**不过，他受了那么重的伤，不知道还能坚持多久。**」

杨子藏忍住了心中的想要去说明的想法，自己目前什么都还不懂。

那个罗鑫队长的话虽然残酷，但他说的是对的，在这里，首先要学会的是理智。

自己哪怕想要帮助这些人，也要有足够的实力才行。

不过...

[**我选择了长期投资，时间直至月底，时序减少 100N。**]

杨子藏决定再给梁喜一百年的寿命保底。

忽然，几人感觉到一股阴冷气息自遥远地方弥漫开来，扩散天地。

地面隐隐震动，四人回头望去，只见上一个地铁站所在之处，出现一道浓烈的黑色喷泉般的东西，它顶天立地，宛如四五十层高楼。

怪物通体张开无数双赤红色的眼睛，扫视著周围。

「**是那个上一层地铁站孕育的怪物！**」

众人心惊胆颤。

没想到这怪物如此恐怖！ 巴士跟它相比，只是小巫见大巫。

杨子藏瞬间出现个感觉，要是自己在售票厅培育Moexl，百分之百会被那怪物吞噬意志！

「**快跑！**」

「**被它发现铁定会死。**」

四人大惊，再次充满慌张情绪的奔跑起来。

......

「**姚队长，不知道那位孕育出四边形Moexl的人，会不会来。**」

宛若堡垒般高大厚重的岩石房屋中，一位身穿小西装，打扮得宛若职业女性的靓丽女生抱著本资料夹，望著门外的道路，有些忧心的道。

「**雪美，你已经问了好几遍了，虽然他是四边形，但也不用一个小时问四遍吧。 还是说，你还想著那个赵安龙？**」

「**行了，赵安龙如果能成功回来，是我们的绝大助力。**」

姚启明站起身来，他的黑瞳宛若墨玉般璀璨光洁，精光汇聚。

在他身旁，另一个稍微有些胖胖的女孩坐在椅子上，拿手撑著脸颊，略显无聊地盯著外面的流云，刚刚就是她怼了赵雪美一句。

姚启明道：「**在那种强敌环绕的情况下，楚强还敢动用一千年的时序为他投资，就是为了表达我们的善意。 那个小伙子也没有辜负我们的厚望，不愧是四边形，天生就比大部分人多一两倍的时序增幅，唉。**」

赵雪美露出思索神色，说道：

「**从他的几次选择看来，这人的人品还是不错的。 哪怕不来，我们至少也会与他结下一个善缘。 到时候让楚强多跟他联系联系，告诉他一些变强的方法，没准他会同意去探寻下安龙哥的下落。**」

她目光沉著地看向脸略圆的女孩，声音清澈如水地道：

「**冯彩琳，没有安龙哥，就没有今日的猎怪者联盟，不要因为他仅是失去消息半年，就忘了他曾经对我们做出过的贡献。**」

「**好好好，我的错，你继续问吧，希望在第十遍前，那个四边形能来。**」

话还没说完，她见到雪美脸上露出惊喜的表情。

「**难道来了？**」

冯彩琳也站起身到门口望了一眼，正如楚强所说的，果然是四个人。

这让屋子内几人都惊喜不已，姚启明也松了一大口气，只要这小子能来，就已经成功了一半了。

......

杨子藏随著记忆中的左明路地图奔跑而来，途中见到了许多地球上不存在的奇怪区域，就在身后不远处，便有栋漆黑的古老高楼。

那栋漆黑高楼就像是中世纪的建筑，却又比那个年代的建筑更为高大、厚重，散发著浑厚的气息。

姚启明等三人一起迎了出来。

「**这里就是猎怪者联盟？**」

杨子藏望了眼这四层高的建筑，它的三层与四层缩小了一圈，在第二层露出圈大阳台，岩石围墙还做出了城垛的形状。

「**欢迎你们的到来。 没错，这里是我们队伍的驻地。 目前大伙儿都外出了。 等晚上，咱们一起为你庆祝庆祝。**」

「**欸，在，庆祝，之前...**」刘明划动了下干涩的喉结，吃力道：「**咱们这儿，有没有，水...**」

......

半小时后，杨子藏在二楼的平台见到了刚从第四层楼下来的瘦高青年。

「**终于见到了，能孕育出四边形Moexl的你。**」

这青年虽然干瘦，但筋骨看上去充满力量，声音有著不合年纪的沉稳，仿佛经历过许多事情。

他充满笑意的道：

「**容我自我介绍一下，我名楚强。 在来到这里之前，是个电路设计师。 现在是团队的后勤人员。**」

「**你好，我是杨子藏。**」

两人伸出手握了下。

「**如果没有你的一千年的说明，我恐怕已经死了，我由衷的感谢你的支援。**」 杨子藏道。

「**不客气。 作为新人，你能做到这种程度才真是出乎我的意料。 要不是因为总部的约束，我还真不会撤走那一千年的时序。**」

「**总部？**」

「**猎怪者阵营总部。 实际上，呵呵，算了不说这个。**」

杨子藏点点头，「**你们也有你们的难处，我知道的。 我也没有真的杀掉他。**」

只要不处于兴奋状态，心间那Moexl蕴含的负面情绪对自己的影响并不算太大。

此刻杨子藏想到自己当时对白鸳的强烈杀意，心中也有些担忧，它对自己情绪的影响是如此的强烈，要是以后的战斗中被影响，很容易出现不理智的情况。

杨子藏抬起头看了眼天际苍穹，又看向身前之人。

「**这到底是什么地方，你们清楚吗？ 我在路上见到许多城市中不存在的建筑，还有地形。**」

楚强道：「**我猜，这里已经不算是地球了。 至于是哪里，我们并不是很清楚，但应该会在未来逐渐知晓答案。**」

「**为什么如此肯定会逐渐知道？**」

杨子藏敏锐的发现了楚强话中透露出来的资讯。

他笑了下，说道：

「**有途径可以逐渐解密，如果你加入我们猎怪者联盟，我会给你分享这些资讯。 而且，还有很多东西可以与你共用，比如能诞生更大力场的资讯，比如武器，以及疗伤的物品等等。**」

他眼神灼灼，望向杨子藏，等待著年轻人的答复。

「**武器，疗伤物品？**」

「**你可以来跟我试用一下。**」

楚强推开厚重的金属门，屋子中出现一排排货架似的东西，上面放置著各种奇形怪状的东西，有矿石、树木，还有怪物的鳞甲。

这些架子都被一层淡白色的光芒笼罩著。

「**有手枪吗？ 我似乎比较适合远端武器。**」杨子藏看著这玲琅满目的东西道。

谁知楚强却摇摇头，他笑道：

「**我看见你用石子秒杀张飞了，大概是因为四边形的缘故，你的Moexl能力真的很强。**」

「**普通两三道线的地铁站Moexl，最多给自己加加速或是减速对方而已，你的却能影响物品，甚至是好几个其他人。 但，如果你想用子弹达到这那种效果，恐怕会失望。**」

杨子藏心中一紧：「**为什么？**」

「**这样吧，我给你拿个东西你试试就知道了。**」

楚强往更里面的房间走去，他扭开房门进入后，很快拿出一副木制的弓箭，以及一根看上去非常普通的秃头箭苗。

「**用过弓吗？**」

杨子藏摇摇头。

自己对打篮球很在行，但弓箭，除了小时候的玩具弓，真家伙别说摸了，见都没见过。

干瘦的男人将这柄半人长的木制弓箭递给他，并将一只箭苗递了过来。
