# 第30章 百年器成功被抢-1



「**罢了，我果然不是拉下脸求人的人。**」

年轻人的神情恢复冷酷帅气，他轻巧的敲了敲大木箱道：「**老宾，这里有没有其它怪物巢穴？**」

果然多了解点资讯能避免很多损失。 如果贸然使用完整邮徽，却没有时序去扩展梦境时间，损失就大了。

空有宝物而不能使用。

杨子藏深刻体会到了捉襟见肘的感觉。

要是有时序，自己马上就可以去到未知的梦境世界获得新能力了。

「**噗~**」

被酒呛到的洛宾白了他眼：「**呵，换做半天前，我会告诉你没有适合你的。 不过现在，我Moexl刚好发现个 c 级交错地。**」

c 级交错地，也叫做新月级。

这个 c 级交错地的裂缝依旧比较隐秘，它在一处居民楼上的房间衣柜里。

四边形Moexl，还是情报收集型的四边形Moexl，能力果然是出众。

洛宾在没有频晶的情况下，想要找到隐秘的 A 级交错地可能依旧有点麻烦，但对于 c 级交错地来说，轻易就能发现附近的。

这可把只有个钝角的梁喜羡慕得不行。

「**我跟著进去真的没关系吗？**」 少年说到。

这个等级对应的频晶孕育者，便属于他这种仅两条线的人。

洛宾道：

「**免费情报。 如果是两名以上三角形人员，可以不借用频晶就能稳定 c 级空间的部分区域显现。**」

三人站在这个衣柜前，看著里面如大地震后出现的裂缝，它内部正透露出混沌的光芒。

「**所以当然没有问题。 这种地方我去过不下一百处。**」洛宾无所谓的道。

梁喜忐忑的跟著两人进入。

这里面到处都是闪动的各种能量场与烟雾。

重叠的各种混乱影像弥漫，只有脚下数米范围显现比较稳定的画面。

前方的草地在水波般的能量滑过后，又变成了砂砾浅滩。

「**吼~**」

「**呜嗷~**」

空间隐隐震动，不同的场景内出现各不相同的野兽们的身影，它们朝这边看来。

「**这就是各种地形重叠的'交错地'了。**」

洛宾笑呵呵指著变异的野兽。

「**放心，它们出不来。**」

梁喜和杨子藏都是第一次进入这儿，霎时觉得特别新鲜。

那些怪物们只要被其它区域挡住，就无法跨越。

但同时，要不了多久，其中一些影像就会在能量波动扩散后，变成了另外的环境模样。

「**我们需要选择走同样的场景，清理里面的生物，去把它们稳定下来。**」

洛宾介绍道：「**这里森林景象比较多，为节约刷新的等待时间，我们就选择森林好了。**」

三人走向左侧如同全息幕布的三维世界。 那是光线昏暗的森林，其中四只瘦弱的老狼正谨慎冲著他们低吼。

三人如同走入一个森林舞台。

噗！

杨子藏挥动银枪，一枪就戳穿了扑上来的那只体型瘦小的老狼头颅。

[**我收获了 7.1Y。**]

七个月，它们的寿命好短。

习惯了动辄几十年寿命的杨子藏略微吐槽了句，看向了另外几只吓得到处乱窜的灰狼。

这片显现的灰狼场地仅仅两个普通客厅大小，被暂时稳定空间的它们根本跑不出去。

一枪掷出，又一头狼被钉在地面，不断哀嚎。

「**真是好枪啊！ 而且你投得又准，啧啧。**」

「**这头狼你们解决吧。**」

杨子藏对著被长枪订住未死的狼道。

洛宾笑哈哈的给他竖了个大拇指，努嘴示意梁喜上。

梁喜紧了紧手上这把加装了木手把、打磨过尖端的钢条，走上前将它戳进了灰狼的心脏。

「**好轻松就得到了几个月寿命诶！**」

梁喜差点感动到哭。

解决完这几头普通的野兽，又待了两分钟，这片区域突地延展了两米左右。

「**稳定完毕。**」

洛宾看著杨子藏夸道：

「**真快啊，才两分钟，用 c 级频晶来稳固地貌，都差不多要十分钟才行。 不愧是四边形Moexl。 你这四边形真厉害噢。**」

这阴阳怪气的家伙明显是在夸自己，杨子藏无语看了他一眼。

「**好了，这表示几天内它不会变成别的世界了。**」

「**只要继续把森林场景变成这样，最后就真有片森林？**」

「**没错。 另外，这些区域叫做交错幕景。**」

杨子藏点头，这的确像一道道帷幕下的小场景。

百闻不如一见，百见不如一练，杨子藏算是明白这里的规则了。

森林场景时而出现，时而消失。 它们虽然数量多，但有些被一个个舞台般的密草丛、田地、河滩等地貌所隔断。

并且这些地貌不断变换。

「**附近没有森林了。**」

「**我们等等就好，免得稳定了其它地貌，到时候要浪费更长时间等它们失频。**」

三人稍微等几分钟，身旁一处荒地消失，出现了另一片紧挨著的森林。

「**好厉害呀！**」

梁喜看著突然自虚空中长出的，延伸到肩膀的树枝。

「**走吧，继续清理狼。**」

噗噗！

枪影闪动。

数只瘦弱的灰狼倒在三人手下。 这片两个舞台大的地貌清理完毕后，洛宾选了头体格比较拽实的狼将它剥皮切肉，从旁边树上凿来木条升了堆火。

「**小子，我教你烤肉。**」

梁喜点点头。

洛宾的动作熟练，而且他来之前备好了调味包，在这儿给两人好好的露了手。

这就是有个老油条的好处，哪怕是置身于危险之地，有这种人在，也像是出来度假。

烤到一半，附近的小片荒地缓慢变动，成为了新的森林幕景地。

杨子藏抬头一看，它远处似乎有个若隐若现的小木屋。

「**像是有人居住？**」

「**不，这里面不会有人。 如果是人形生物，那么就要小心了。**」

洛宾看了眼，示意梁喜接手自己串的狼肉。 他擦擦手拎起钢条往那里面走去。

杨子藏跟上。

「**呜嗷~**」

密密麻麻的身影自周围黑暗中冲出。

「**这么多？**」 洛宾吓了跳。

杨子藏持枪进入。

[**梁喜想要为我进行主动投资。**]

[**梁喜为我投资了 50N。**]

「**我们已经稳定了两片森林区域，这是第三处。 随著相同地貌稳定越来越多，其它未显现的森林区域会更快显现。**」

「**这里毕竟只是 c 级区域，面积并不大，才清理两个就已经出现地貌聚合现象了。**」

洛宾掌中钢条金色虚影流转，虽然他并没有进行身材管理，但能在此摸爬滚打三年的人显然不是废材。

面对猛然扑袭而来的灰狼，他身形灵动，出手既快且准，几乎在电光火石间便精准地终结了近身两匹恶狼的性命。

噗、噗！

杨子藏手上长枪舞动得虽如同小孩打架，却速度奇怪，看上去倒也有几分像模像样的威势。

[**我共计收获了 2.11N。**]

两只狼倒地。

算上之前的两小块地貌区域，这一趟已经收获了接近五年的时序。

这就是这里大多数人的时序来源。

看上去似乎不错，却只有半个月的时间来累积它们，而且还要给月末留下三百年。

进入这里的普通人面对这些凶猛的野兽们，要每天至少要获得二三十年额外时序才行。

这压力可想而知。

「**太多了，我一个人去好了。**」

[**投资人洛宾想为我进行主动投资。**]

[**洛宾为我投资 100N。**]

进行四倍增幅保护后，杨子藏拎著枪冲入十几米外区域。

狼群早已等待多时，数道凶恶的身影扑了上来。

枪影闪动，这些狼虽然数量多，但面对近在咫尺的杨子藏，它们的爪子却总是从他身旁惊险划过，落不到年轻人身上。

噗嘭！

一具接一具的灰狼躯体倒在血泊中，鲜血染红了地面。

它们就像主动撞到枪尖上。

洛宾在后方看得一愣一愣的。

清理完接近二十只狼的杨子藏长松了口气。

这片区域面积已经相当于之前的两处森林区域的总和。 随著他的前进，小木屋已经在不远处。

摸到这扇木门时，杨子藏想起了之前的生锈铁门。

「**应该没这么倒楣吧。**」

他还是有点不放心，于是围著木屋转了圈，从一角因为漏雨发霉变形的木板处，找到了个能看清里面的缝隙。

凑眼看去，屋内是荒废许久的模样，里面有著猎人准备的物资，绳索、斧头与钉子，以及挂在墙上的干瘪狼皮和一张硬弓。
