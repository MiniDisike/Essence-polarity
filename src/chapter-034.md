# 第34章 求你抢回来-1



[**王总为我投资 4000N.**]

｛**杨子藏，快拿回你的武器。**｝

王育安超大声的在心念中响起，他焦躁不安大吼，近乎气急败坏。

｛**你要卖就卖给我！**｝

｛**你根本不懂时序战器的价值，你甚至可能都不知道增幅时序被抵消后会有两天才能恢复，而时序战器随便一挥动就能造成数十上百年时序打击！ 你这个傻叉！**｝

「**啊？ 真的？ 你们干嘛不早点说啊，他们都走了。**」

｛**他们连门都还没走出，快去啊！**｝

「**这，不好吧，我话都说出口了。 这不是不讲信用吗。**」

｛**我去你大爷的。**｝

这话让某些人的肺都快炸了。

｛**你脑袋里被塞大便了吗，是不是有病啊，你...**｝

杨子藏遮罩了他的心念通讯。

没想到被抢劫还能有这种好处，杨子藏欢喜的感应著这堆慢了步才投资过来的数千年寿命。

骂归骂，王育安这家伙并没有抽资。

可以啊这小子，这么命长，不愧是玩投资的。

不晓得这单洛宾会收王育安多少时序。

前方，晃著光膀子的洛宾早已经嗷嗷叫著冲了上去。

「**谁也别想拿走，这是我们的武器。 杨子藏是我的队员，武器是我们的！**」

手上带著缕红色的金芒让那七人面具下的神色大变，纷纷躲避著胖子的追赶。

余晨虽手握长枪，但一百年时序跟一剑戳来就会要人老命的普通钢剑相比，却根本不敢上前。

正在这时，众人要出去的门突兀灰暗起来。

「**不好，这片空间被晶石同频了！**」

杨子藏看向身旁歪著脑袋打量著自己神色的梁喜。

「**同频是啥意思。**」

「**有人用 c 级交错频晶，将这里绑定了的意思。**」

远处的燕逸通过感知画面看著新进入的那群蒙面人。

螳螂捕蝉，黄雀在后。

他们带著新月级频晶，一进入就将裂缝给封闭了起来。

没有手持晶石之人的允诺，将无人能够在稳定这片交错地前离开此处。

「**有这个东西，我们就可以关门打狗了，哈哈哈！**」

「**兄弟们，给我上！**」

乱糟糟的混乱人群瞬间就战斗到了一起，而手握长枪的余晨则是首要目标。

「**靠，这武器根本无法注入时序。**」

嘭！ 两道身影从人群跃出，其中一人手持冒著浓烈金色虚影的铁棍，一棍子敲在他手臂上。

余晨面色一变，这一棍居然带著超过五千多年的时序增幅。

刚刚保下手臂的他还来不及有所应对，另一道高大身影已经落在他身旁，一记带著金色虚影的上勾拳就向著下巴砸来。

「**啊啊！**」

有过无数战斗经验的余晨仓皇后退躲过。

他抬起长枪挥动，而对方明显是受过更专业格斗训练的人，来人冷哼一声，一脚踢开武器，双手快速砸了过来。

嘭嘭！ 两拳下去，余晨双臂被打爆，痛苦的后仰倒地，

「**得手！**」

来人明显是这只队伍的首领，声沉如钟，充满威严。

「**武器还来！**」

洛宾持钢剑自混乱人群中开辟出条血路，此刻上面的红芒已经转为纯金色，说明时序已经降到了一万年以下。

「**嗯？ 时序战器被掌控了。**」

蒙面首领抬头注视向杨子藏，其目光冰冷，仿若看待只可随意拿捏的牲畜。

「**老三，拦下这家伙。**」 他瞟了眼洛宾沉声道。

「**是！**」

队伍中一人迅速后退，他的Moexl快速展开后，一个黑乎乎的物体啪地声砸在洛宾身上。

洛宾灵敏的身姿顿时如被什么刺中般，手中长剑啪嗒落在地上，额头顷刻间就渗出虚汗跌倒在地。

他捂住胸口，看向蒙面之人。

「**洛宾，你怎么了？**」 杨子藏急道。

「**我的，心脏.... 这能力... 你们是... 你是... 赤魈宫的宫主，李王？**」

「**居然认出来了，没错，我就是宫主。**」

洛宾神色一变：「**果然是你们... 我的心脏状态.... 被调转了，这些家伙，你们肯定在哪里囚禁了其他人，把我们的状态对调了。**」

周围包括余晨在内的几人微震。

联想到之前这帮人的对手都突然莫名奇妙重伤，他们这才惊觉是什么原因。

居然是囚禁人后将其弄伤，在关键时刻让这人的状态与别人对调。

那身材高大的蒙面人将脸上面具一扯，露出下方不怒自威的脸庞。

「**是又如何。**」

他将手上银色的时序战器随手一丢，身边自有人接住。

「**杨子藏，马上解除时序战器的控制权，否则我就亲自动手，送你归天。**」

李王身上散发著如有实质的压迫力，踏上剧台走向楼梯。

杨子藏眼睛微眯，看来今天这武器还真不好送出去。

他深吸口气，看向下方明明赤手空拳，却散发强大威胁之人。

这所谓的宫主胸口白色环形再次显现，双拳也随之覆盖上了金色的虚雾般的焰火。

「**主动解除，或者死后，被动解除。**」

李王声震这片空间。

杨子藏侧伸手臂，目光直视楼下之人，轻声对梁喜道：「**武器借我一用。**」

感受到手上的木柄触感，他鼻息喷出两道浊气。

竖起这把钢条打磨出的利剑，胸口圆圈内的四千年数位顷刻间归零。

四千年，四倍增幅。

杨子藏一个纵跃而起，身体在半空时，双手握住的长剑冒出红雾般的虚色火焰。

「**哈啊！**」

下方的宫主李王脸上顿时露出残忍笑容，他双拳的金色虚影内部突然冒出缕红色丝线。

「**敢来这里，我怎会不做应对。 来吧，来场男人之间的战斗！**」

嘭嘭嘭嘭！

杨子藏落地瞬间，他步伐稳健迅捷的欺身上去，双拳打出残影，空气都发出爆鸣，势如山崩。

李王的拳势如山呼海啸。

「**喝啊！**」

杨子藏侧身躲过必中之拳，他如一叶扁舟般在拳影中腾挪。

「**怎么会。**」

李王的拳头每每要在砸中他的瞬间，从起脸庞、头发、胸侧差之毫厘的打过。

这场战斗看得众人屏住呼吸，杨子藏居然能应对这样的武道大师！

当他扯下面具的那一刻，众人都已经确定了他的身份。

把队伍名起得宛若古代门派的李王，正是现实世界中的一位拳道高手，曾拥有自己的格斗俱乐部。

在这个人人都感到难以应对的世界中，他反而是喜欢上了这里。

此处，是正他这样崇尚武道，能以格斗技来获得利益的好地方。

在这一年之间，他已击败过数名曾经在这片区域驰骋无忌之人，获得大量时序，也笼络了批喜好此道的追随者。

如今他自封为宗门之主，打算开宗立派。 连真名都做了更改。

「**时序战器落入他们手里，恐怕总部都难以拿回去了。**」

「**就看他能不能杀死杨子藏，如果能够杀死，那么总部就有理由惩戒他了！**」

「**呵，惩戒李王？ 那恐怕又是番浪费时序的战斗，总部那些人才不会浪费时序呢。**」

[**我获得了杨千的 1000N 投资。**]

「**杨千？ 好家伙。**」

杨子藏生怕那姓王的抽资，那今天自己就算是交代在这里了。

此刻居然有其它人给自己千年投资，有这些时序在手，心中总算能稳当了点。

只是没想到这人居然会是被自己胖揍了顿的杨千。

难道我还有点搞推销的潜质？ 他不禁想到。

｛**杨子藏，你若愿意将武器卖给我，我可为你投资五千年，助你活下来。**｝

一个陌生声音在心灵链接中主动说道。

｛**我也是，杨子藏，你若愿将武器卖给我，我为你投资两千年全权时序，时间一年，随你挥霍。**｝

｛**怎么样杨子藏，我能比他们出更多，只要你愿意，时序马上到账。**｝

七嘴八舌的声音在心灵连结中喊话，却没一个人真的给出投资。

「**呵。**」

拳影下的杨子藏边躲边退，还好只是这一个人，要是再来个这样的高手，自己肯定来不及腾挪。

这人可不比那些从地铁上冲下来的被洗脑者们。

哪怕有自己的能力去改变他的拳势轨迹，但他却在几秒后就发现了这点，正在用双拳封锁自己的移动范围，并不断用身体将自己逼到逼仄角落处。

而自己虽然手持武器，却找不到一个可以刺出的机会。

这就是面对武道大家！

自己哪怕可以用能力去调动对方的部分挥拳轨迹，但面对这种随意出手都一招一式之人，自己根本没百分百把握能够一击必中。

哪怕是上万年的时序增幅，居然也不允许自己能够有所失误，一旦抵消，自己就会陷入万劫不复之地。

「**可恶啊。**」

于此同时，燕逸已经从杨子藏的心灵连结中退出，与在现场的同伴取得联系。

「**老弟，hold 住吗？**」

[**放一百个心。**]

童文奇一直躲在众人身后。

另一伙人得到时序战器后，就将洛宾围了起来，用这把武器当做棍子抽打著洛宾。

「**废物啊，这武器就是你的又如何？**」

「**你说是你的，就叫它不要抵扣你的时序啊！**」

一击一百，数千年时序也只能挨几十下。

洛宾的蓝色软帽早已被打飞，骇人的伤口看上去有几分狰狞狞。
