# 第36章 念力寄往邮徽世界-1



想到强者，他不由自主想到所谓的「**总部**」。

只因有所谓雕像的东西，控制了那里的人就几乎等同于这个城市的主宰。 许多能力都要雕像才能运用，要是自己能去到拥有雕像的场景就好了。

他只是这么一想，根本未曾指望能去到类似雕像所在的地方。

正当杨子藏继续在脑海中冥思，该去往什么样的梦境世界才能获取最大利益时，忽然手上由Moexl光线所构成的方框扩展变大，成为一个信封。

[**我想前往可以获得格斗技与雕像世界，是否要邮递至这样的意念时空？**]

什么！

杨子藏一下坐了起来，他看向自己手上的那「**光线信封**」上，封面出现了扭曲的陌生文字。

「**难道真的可以... 去到拥有总部那种雕像的世界？**」

他的心跳不由得加快。

想到这里，杨子藏干脆在心中多加了几个要求。

自己还有哪些问题可以一并解决？

治疗药剂、更多时序、更强的力量...

真要去探究那可就太多了... 他想明白这个世界是怎么回事，想知道雕像是什么意思，还想获得能在这个世界立足的高端能力，以及回到地球...

杨子藏想了一大堆，看著信封上的片组越来越多，他按奈住心中的渴求。

「**不能加这么多条件，绝对会出问题，到时候可能什么都得不到。**」

做出取舍后，他轻声对著自己道：

「**我想去往一个，跟这个世界核心秘密有关的，同时跟雕像有关的，能获得这个世界高端能力的，相对安全的意念世界。**」

信封上的片语随著他的话音落下，减少了许多。

[**是否要邮递至这样的意念时空？**]

「**是！**」

[**邮递费用，1R。**]

这么便宜？

杨子藏还来不及多想，原本就光线阴暗的房间变得混沌起来，如同旋转的灰雾迷泽。

手上光芒信封化为意识漩涡，巨大的引力似乎让灵魂离体般被卷入了里面。

顷刻间，杨子藏感觉自己意识被吸入，天旋地转。

梦幻般的声音若有若无自意念深处响起。

[**失落王庭，青白之星，为您服务...**]

[**我消耗了 1R 时序本金。**]

[**我消耗邮递次数1，剩余2/3。**]

......

空气中弥漫著浓郁的草木味。

阵阵鸟叫声中，杨子藏缓慢睁开了眼睛，他发现自己躺在古树横生的丛林。

[**来自青白之星的提示：在此，您可以加速时间。**]

天青云朗，绿树成荫，清风如轻纱摩挲般舒润的抚过皮肤。

「**比上次更真实，这里... 不像梦中似的。**」

身穿染血短袖衬衫的他刚站起身来，忽然一片阴影投下。

轰~

狂风席卷，紊乱狂暴的气浪将杨子藏掀入旁边荆棘林中，皮肤瞬间被刮伤。

长著折翼的灰褐色生物冲入高空，飞向远方。

满脸不可思议的杨子藏从荆棘中爬出，跌跌撞撞起身追去。

「**那是... 翼龙？！**」

他太过震撼，内心不安的看著翼龙消失在远方。

「**有恐龙？**」

杨子藏扶著树干望向山下。

巨木参天，茫茫天地四野无人踪。

远处，更多类似翼龙的生物翱翔山巅。

「**老天，这像是有雕像的样子？ 搞错了吧！**」

夜晚，探寻了周围区域的杨子藏见到了更为古老的生物，有些看上去极其恐怖，比如林中长著胡须的巨蟒。

噼啪作响的篝火旁，杨子藏手指一捏，光芒般的信封出现。

「**你把我送到哪儿来了。**」

这东西还能让自己来这里两次，也就是说它一共可以让自己进入这梦境三次，现在是第一次。

除了身上的衣物，只要是睡觉前带著的，目前都出现在了这个世界，包括那把从 c 级交错地寻来的弓与几只箭。

这算是意外之喜，弓是梁喜顺手放在睡觉的杨子藏身边的，毕竟这种武器对那孩子来说毫无用处。

「**可惜了梁喜的那把剑，早知如此，该拿著睡觉的。**」

食物的香气传来，杨子藏拿鼻子嗅了嗅，从篝火旁的灰烬中刨出两个褐色的瓜果。

这是他下午找到的类似地瓜般的食物。

跟上次的梦境一样，在荆棘林中受伤的皮肤几分钟便复原，而且哪怕是走了半天，自己也根本不渴不饿，所以这真的依旧是在梦中。

在这里虽然不需要吃食物，但同样的，自己可以无限的享用各种美食而毫无顾忌。

这算是在意识世界内为数不多的消遣。

周围野兽的嚎叫隐约传来，杨子藏抬头望去，刚好月上中天。

[**我消耗了 0.12R 时序本金。**]

正拨开个「**地瓜**」皮的杨子藏听到了Moexl反馈的心念之声。

他的眉头一皱。

自己进来的时候是梦境世界的正午，到现在刚好十二个小时。

「**原来所谓的消耗时序是这个意思，在这里停留的时间长短，是由时序的多少来决定的吗？**」

如果真是如此，那自己便能真正体会到什么叫做黄粱一梦。

「**乌噫，乌噫！**」

林间响起无数脚步声，一个个嘴唇前突，浑身长著细长毛发，披头散发佝偻著粗皮身躯的矫健身影从四面八方窜了过来。

其中一些动作灵敏地两三下便上到了大树上，虎视眈眈地盯著下方的杨子藏。

「**哇我，哇哇！**」

他们手上拿著石块和木棒，既畏惧又充满好奇的盯著中间那堆篝火。

「**猿人，野人？**」

咔嚓。树枝断裂。

故意佝偻着背，浑身长着肌肉的人形生物轮廓自这十来名野人身后的黑暗中走出。

他腰间挂着藤曼缠绕的腰带，上面挂着两个小狼头骨。

「**是被它吸引的吧。**」 杨子藏笑了下，将手上剥开的地瓜丢了出去。

调度能力发动，地瓜稳稳当当落在了这人手上，这精准的动作惹起一片呜呜喔哦的叫声。

在这完整邮徽所指引到的世界中，Moexl能力也能够运用，杨子藏心想，这或许就是自己所想的「**安全世界**」的回应。

有Moexl能力，自己的确是安全许多。

但也可能在完整邮徽所到达的世界，所有人的Moexl都能够使用。

身材魁梧的野人将烤地瓜拿在爪中嗅了嗅，犹豫良久后，在其他同类「**唔唔**」的催促下，他疑惑的撕下小块塞入嘴中。

烤熟的地瓜入口既化，魁梧野人如遭雷筮般瞬间呆立，双目圆瞪。

他不敢置信的嚼了下。

「**嗷~**」

野人首领猛地原地绕圈窜了起来，双拳捶著胸膛唔唔嚎叫，他似乎是平生第一次吃到如此美味的食物。

扯下半个地瓜自己吃后，他将剩余的半个地瓜丢了出去，其它几名野人当即抢夺起这块地瓜，场面混乱无比。

「**唔喔，唔喔噢噢噢！**」

杨子藏笑著看著它们，但马上他就笑不出来了。

尝到烤地瓜的众野人食欲大增，它们虽然依旧恐惧于那堆篝火，但却多了份敢于靠近它的理由。

他们举起了石头和木棍，喉咙如野兽般发出威胁的声音，半弓著腿一步步靠近杨子藏，因为那里分明还有好几个这样的东西。

「**野哥，地瓜而已欸。**」

果然「**把别人的东西收归己有**」是刻在人类 dNA 深处的本能，无论在什么时代。

杨子藏拿起弓箭起身，在他们抛出手上的石头砸中自己前，胸前Moexl浮现。

[**时间加速，前往一日之后**]

几个石块划过空间照著杨子藏的身体而来。

[**按照当前脚程，二十四小时我可以前行百里。**]

「**先暂时离开点距离便好。**」

杨子藏随意选了个方向，使用了时序。

他从容迈步的瞬间，天地似乎被按下了加速键，身体嗖地消失在一步之外。

石头们噼啪打落在他原先所在的位置上。

众猿人瞬间惊呼起来，慌成了一片。

......

光芒亮起又转黑，一秒之后，杨子藏已经来到了第二日的午夜。

[**我消耗了 1R 时序本金。**]

这便是在此处多出来的能力，时间加速。

「**还好不用真的当个旅行家。**」

杨子藏放心了不少，随后他就诧异的发现自己在黑暗中还能稍微看清点东西。

这应该是第二天午夜。

转头一看，远方浓烈的火光在山头熊熊燃烧，宛若广袤黑夜中的神明，如此特立独显。

「**好家伙，该不会是我刚刚烤地瓜的地方吧。**」

「**还真是。**」

重新回到此处的杨子藏无奈的看著这熟悉的地形。

生火时明明已经清理出了安全的区域，能搞出这动静，毫无疑问是那些类似猩猩姿态的人类所干的了。

它们大概是在掏地瓜的时候，将篝火掀得到处都是，惹燃了山林。

火光中好些动物被烧死，还好这火势只在这山头附近。

忽地，周围树上响起几声野人的唔喔叫声。

随著此起彼伏的叫声响起，一个浑身烧得秃一块、红一块的倒楣蛋哀嚎冲来趴在地上叫唤。

杨子藏停下了正打算加速时间离开的脚步。

越来越多的野人从树上跳下，或者从旁边山林中窜出，趴在地上。

他们不住将头抬起看杨子藏一眼后，又埋下抵在地上，浑身颤抖，很是恐惧。

这是它们表达自己臣服的动作。
